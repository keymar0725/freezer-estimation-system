# フリーザー工事 原価積算システム (Freezer Estimation System)

このプロジェクトは、プラントエンジニアリング（フリーザー設置工事・配管・電気工事など）における複雑な積算業務を自動化するためのPythonプログラムです。

## 🌟 特徴 (Features)
- **フリーザー仕様からの自動計算**: 機種、寸法、庫内温度から必要パネル面積や仕様を自動算出します。
- **水冷式配管の自動サイジング**: 流量や直管距離、配管ルートの複雑さから最適な配管径と相当長を自動計算し、ポンプ等を選定します。
- **Excel連携**: 使い慣れたExcelをインターフェースとして使用し、最終的に「工事原価表」を出力します。
