# フリーザー工事 原価積算システム (Freezer Estimation System)

本システムは、プラントエンジニアリングにおける複雑な見積・積算業務を自動化するためのPythonツールです。

## 🌟 主な機能 (Features)
- **自動積算エンジン**: 入力シートの仕様（フリーザー型式、寸法、庫内温度等）から実質パネル面積を自動算出。
- **DB連携**: 「機械設備工事積算実務マニュアル」のCSVデータと連携し、該当する材料単価・労務費を自動引き当て。
- **配管サイジング・圧力損失計算**: 水冷式冷却水系統において、配管径の自動選定およびポンプ全揚程の算出を実施。
- **工事原価表エクスポート**: 計算結果を所定の「工事原価表」フォーマット（Excel）へ自動転記し、別名保存。

## 🛠️ 必要環境 (Requirements)
- Python 3.8+
- ライブラリ: `pandas`, `openpyxl`

## 🚀 使い方 (Usage)

### 1. 準備
リポジトリ直下に以下のファイルを配置してください。
- `積算入力シート.xlsx` (入力用テンプレート)
- `機械設備工事積算実務マニュアル*.csv` (DBとして使用)
- `01-2 工事原価表.xlsx` (原本ファイル)

### 2. 計算と出力
ターミナルまたはコマンドプロンプトから実行します。

```bash

python estimation_engine.py
